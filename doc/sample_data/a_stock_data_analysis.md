## 🎯 系统设计启示

### 1. A股配置策略
```yaml
# A股市场配置
a_stock:
  indicators:
    structure: "wide"
    fields: ["净资产收益率", "基本每股收益", "销售毛利率", "资产负债率"]

  balance_sheet:
    structure: "wide"
    key_fields: ["*资产合计", "*负债合计", "*所有者权益（或股东权益）合计"]
    detail_fields: ["货币资金", "存货", "固定资产", "应收账款"]

  profit_sheet:
    structure: "wide"
    key_fields: ["*营业总收入", "*净利润", "*营业总成本"]
    detail_fields: ["营业收入", "营业成本", "销售费用", "管理费用"]

  cash_flow_sheet:
    structure: "wide"
    key_fields: ["*经营活动产生的现金流量净额", "*投资活动产生的现金流量净额"]
    detail_fields: ["销售商品、提供劳务收到的现金", "购买商品、接受劳务支付的现金"]
```

### 2. 查询优化策略
1. **分别查询**: 三个财务报表分别查询，避免数据冗余
2. **指标优先**: 优先使用财务指标表进行快速查询
3. **详细分析**: 需要详细数据时查询具体财务报表
4. **时间序列**: 支持27年历史数据的趋势分析

### 3. 缓存策略
- **分层缓存**: 财务指标小数据集频繁缓存，财务报表大数据集按需缓存
- **增量更新**: 利用季度更新特点，实现智能增量更新
- **压缩存储**: 对历史数据进行适当压缩，提高存储效率
