| 项目 | 关注点 | 处理方法 | 备注 |
| :--- | :--- | :--- | :--- |

## 三、 “资产负债表”及相关附注

### 3.1 资产类项目

| 项目 | 关注点 | 具体审查内容 | 处理方法 | 备注 |
| :--- | :--- | :--- | :--- | :--- |
| **货币资金** | 安全性 | 现金及现金等价物÷有息负债＞1，可适当放宽为：(现金及现金等价物+可迅速变现的金融资产价值)÷有息负债≥1，或(现金及现金等价物+可迅速变现的金融资产价值)÷一年内到期的有息负债＞1 | 安全性分析，满足则基本具备及时偿还短期债务的能力，可以和历史比较，但无须和同行比较 | 可迅速变现的金融资产有交易性金融资产、银票等 |
| **货币资金** | 异常信号 | 货币资金余额比短期负债小很多 | 危险和警惕信号 | 警惕控股股东与银行签订集团资金管理协议或现金业务管理协议 |
| **货币资金** | 异常信号 | 定期存款和其他货币资金数额很大，但大部分是使用受到限制的货币资金或其他货币 | 危险和警惕信号 | 警惕控股股东与银行签订集团资金管理协议或现金业务管理协议 |
| **货币资金** | 异常信号 | 其他货币资金数额巨大，缺乏合理解释 | 危险和警惕信号 | 可按货币资金平均余额为本金计算的应有利息模拟 |
| **货币资金** | 异常信号 | 货币资金产生的利息收入，显著低于常见理财和存款利率 | 危险和警惕信号 | 可将财务公司开出的银票当作商票看待 |
| **应收票据** | 商票占比 | 应收票据里有大量商票 | 警惕公司收入的真实性 | |
| **应收票据** | 商票占比 | 公司应收票据一直是由银票组成，某年突然大量出现商票 | 警惕公司遇到销售困境，开始放松销售政策 | |
| **应收票据** | 与经营规模不匹配 | 非金融企业报表存在与正常经营规模不匹配的应收票据 | 警惕公司不务正业 | |
| **应收账款** | 增长过快 | 应收账款大幅增长，增长幅度超过同期收入增幅 | 警惕公司临时放松销售政策，或提前确认收入甚至虚构收入 | 如果原基数过小，可忽略 |
| **应收账款** | 占比过大 | 应收账款占营业收入的比例较大，且有大部分（如超过三成）是一年以上的应收款 | 警惕 | 可将商票余额也计算为应收款余额 |
| **应收账款** | 占比过大 | 所有带“应收”两个字的科目总和，减去银票金额，看其占总资产比例是否过大，一般超过三成已经算严重，过半显然有问题 | 警惕 | |
| **应收账款** | 异常情况 | 应收账款余额÷月均营业收入，思考是否过大，进行同业对比，处于中位数以上，可暂时认为偏大 | 警惕 | |
| **应收账款** | 异常情况 | 某些时间很长的大额应收账款，突然付款了，在这笔款项支付的前后，新增了其他累计金额近似的异常支出 | 警惕 | |
| **应收账款** | 异常情况 | 某些应收款单独测试减值准备 | 警惕 | |
| **应收账款** | 异常情况 | 应收款项集中在少数几家关联企业 | 警惕 | |
| **应收账款** | 应收账款周转率 | 进行历史对比和同业对比 | 营运能力分析 | 应收账款周转率=营业收入÷平均应收账款 |
| **应收账款** | 应收账款周转率 | 显著低于行业平均水平 | 警惕并分析原因 | |
| **应收账款** | 应收账款周转率 | 呈明显下降趋势 | 警惕并分析原因 | |
| **预付账款** | 大幅增长 | 预付账款大幅增长，尤其是预付工程款、预付专利技术或非专利技术的采购款大幅增长 | 考虑是否存在通过预付款流出资金，然后再以营业收入回流，美化财报的风险 | |
| **预付账款** | 异常情况 | 长期挂账的预付款 | 警惕资金被挪用（超过一年，大概率有问题） | |
| **预付账款** | 占比大幅波动 | 预付款占营业收入或营业成本的比例，发生大幅波动，缺乏合理解释 | 警惕并分析原因 | |
| **其他应收款** | 异常情况 | 剔除“应收股利”和“应收利息”后，数额较大 | 危险和警惕信号 | 阅读2018年之前财报，无需剔除“应收股利”和“应收利息” |
| **坏账准备** | 计提标准 | 同行业不同公司进行比较 | 注意不同坏账计提规则对利润的影响 | |
| **坏账准备** | 计提标准 | 变更坏账计提标准 | 警惕，并思考其原因及影响 | |
| **坏账准备** | 异常情况 | 统一按某固定比例对所有应收账款和其他应收款提取坏账准备 | 警惕 | |
| **坏账准备** | 异常情况 | 某个欠款单位已经明显不可能还款，但公司仍没有计提100%减值 | 考虑所有应收款都存在少计坏账准备的可能性 | |
| **存货** | 减值风险 | 可供出售的存货风险偏高、有快速贬值的特性（新鲜食品、高科技制造），需与同行业其他公司的存货计提的应有计提准备进行比较 | 衡量存货价值是否真实 | 存货（含消耗性生物资产）计提的减值准备，当减值因素消失时可转回利润表，其他资产减值准备不允许转回 |
| **存货** | 大幅增长 | 存货大幅增长，且幅度超过同期营业成本的增长 | 警惕并分析原因 | |
| **存货** | 存货周转率与毛利率 | 存货周转率明显下降，却同时伴随着毛利率显著提升 | 危险和警惕信号 | 除非产品提价，或生产成本下降（需了解同行业是否有此水平的下降） |
| **造假的高危地带** | 4种情形 | 持有大量消耗性生物资产的农、林、渔、牧公司 | 谨慎涉足 | |
| **造假的高危地带** | 4种情形 | 税收优惠多，造假成本低 | 谨慎涉足 | |
| **造假的高危地带** | 4种情形 | 产品不易核查、监控、清点 | 谨慎涉足 | |
| **造假的高危地带** | 4种情形 | 产品差异大，很个性化，不易找到价格参照物，价值判断难度大 | 谨慎涉足 | |
| **存货** | 本期大额减值，下期卖出获利 | 对联合营公司股权，通过超额计提减值准备，降低当期利润，甚至造成巨额亏损，然后下期再按照公允价值出售，增加利润及利润的同比增速 | 警惕操纵报表 | |
| **长期股权投资** | 同一控制 | 导致你怀疑是双方可能属于同一控制 | 警惕操纵报表 | |
| **长期股权投资** | 通过小额交易，改变股权类别 | 在股权归属的会计类别边缘，进行小额的股权交易，改变资产分类，并因此产生了较大投资收益，则需要怀疑这笔交易是否真实、转让作价是否合理、公允价值是怎样计算的 | 警惕 | |
| **长期股权投资** | 将一项交易分拆成多笔 | 公司对某一绝对控股子公司，分多次购入成本，其在导致资产分类转回的临界点附近连续交易，且多笔交易作价有明显差异 | 警惕 | |
| **投资性房地产** | 计量规则 | 注意是以成本模式还是公允价值模式计量 | 以成本模式计量的投资性房地产，可能是隐藏的资产 | 公司利用转换计量规则增加的收益，通常调整股东权益，一般不计入利润表 |
| **固定资产** | 变更折旧政策 | 公司突然改变折旧政策，须搞懂公司为什么改，改得合理与否，改了会带来什么利益 | 警惕 | |
| **固定资产** | 账面价值极低 | 有些固定资产年折旧，账面价值极低，但事实上，它们的实际价值没有那么大降幅，或者还在增值 | 可能是隐藏的资产 | 如：水电企业的大坝、白酒企业的酒窖、企业自用的城市中心区房地产 |
| **固定资产** | 固定资产周转率 | 进行历史和同业对比，了解公司资产使用率、反映公司属于“轻资产”还是“重资产”公司 | 营运能力分析 | 固定资产周转率=营业收入÷固定资产净值 |
| **在建工程** | 迟迟不转固 | 在建工程进度为100%，但迟迟不转为固定资产 | 警惕并寻找原因 | |
| **在建工程** | 不断追加投资 | 在建工程在前期规划金额之外不断地追加资金 | 警惕并寻找原因 | |
| **在建工程** | 大幅增加 | 在建工程大幅增加，尤其是和生产经营规模和发展规划不相适应地大幅增加 | 警惕并寻找原因 | |
| **在建工程** | 异常情况 | 在建工程投入资金数量的增加，在建工程进度没有同比例增加，反而停滞甚至减少 | 警惕并寻找原因 | |
| **在建工程** | 异常情况 | 在建工程刚转为固定资产不久，又需要大修或技改 | 警惕并寻找原因 | |
| **开发支出** | 资本化合理性 | 开发投入资本化的合理性 | 需在理解行业和公司的基础上独立判断 | |
| **商誉** | 大额商誉 | 经常进行收购，并产生大额商誉 | 需关注：1）收购作价是否公允；2）尝试将商誉还原成资产，并按照资产折旧规则，重新考虑公司价值；3）去掉商誉后，公司的负债率是否让你担忧 | |
| **商誉** | 减值披露 | 仅披露“经测试未发生商誉减值”或“依照减值测试，计提商誉减值准备X万元”的结论，未按照要求披露商誉减值的测试过程、设定的相关参数及减值损失的确认方法等 | 警惕 | |
| **商誉** | 可疑情况 | 包含大额商誉的收购并出售，其卖方是否与公司有经营活动往来，考虑是否存在将公司投资活动现金流出转化为经营活动现金流入的可能性 | 警惕财务欺诈 | |
| **长期待摊费用** | 数额大小 | 数额越大，说明公司资产质量越差 | 警惕公司不恰当地将费用资本化 | |

### 3.2 负债与权益类项目

| 项目 | 关注点 | 具体审查内容 | 处理方法 | 备注 |
| :--- | :--- | :--- | :--- | :--- |
| **有息负债** | 占比 | 有息负债占总资产比例，和同行，尤其是行业内龙头企业对比 | 大幅高于或低于都值得思考，如果大幅高于，属于比较激进的需小心 | |
| **有息负债** | 利率 | 如果公司借款的代价较高，通常是危机信号，尤其是如果自有货币资金挺多，但仍然以较高利率借款 | 危险和警告信号 | |
| **应付账款、应付票据** | 数额大增 | 需结合对公司的理解去判断，是由于公司在商业链条上地位提升，还是资金链出现问题 | 警惕并寻找原因 | |
| **预收款项** | 数额略高 | 可能是推迟确认收入，隐瞒利润 | | |
| **应付职工薪酬** | 高管和核心人员 | 了解高管和核心人员的履历、薪酬，分析公司核心竞争力 | | |
| **应付职工薪酬** | 员工情况 | 了解员工结构、人员信息，分析公司的扩张和收缩意图 | | |
| **应付职工薪酬** | 员工平均薪酬 | “应付职工薪酬”科目附注的“本期增加”数据，基本上为当年发生的员工成本（会计上的口径），用该数据除以员工总数，计算出员工当年平均薪酬 | 观察是否合理；与往年相比、与同行相比，观察是否异常 | |
| **应付职工薪酬** | 工资黏性 | 如果公司净利润率很低，而职工人数又多，可以做个测试：看人均工资上调多大比例，会使公司由盈转亏 | 如果该比例很小，建议谨慎决策 | |
| **应付职工薪酬** | 提前预提 | 通过应付职工薪酬的预提来平滑利润，体现在财报上：除了应付职工薪酬的余额波动较大，递延所得税资产数额增加也很大 | 可能在调节年度间利润 | |
| **专项应付款** | 良性债务 | 政府给予的、具有专项或特定用途的款项，在没有按指定用途花光之前，余额在本科目 | 隐藏的资产 | |
| **递延收益** | 良性债务 | 企业已经收到或应该收到的、对未来发生的费用或损失的补助 | 隐藏的资产 | |
| **库存股** | 回购并注销 | 对股东有利还是有害，取决于回购价格的高低，需要首先判断公司价值 | | |
| **库存股** | 股权激励 | 期权的公允价值或限制性股票的折扣部分，是员工薪酬的组成部分，根据激励对象所处的岗位、计入相应的费用 | | 相关费用在服务期内按时间加权分摊（前少后多） |

### 3.3 整体分析指标

| 项目 | 关注点 | 处理方法 | 备注 |
| :--- | :--- | :--- | :--- |
| **资产增长率** | 总资产增长率 | 和同行，尤其是行业内龙头企业对比 | 成长性分析 |
| **资产增长率** | 净资产增长率 | 进行历史和同业对比 | 成长性分析 |
| **资产增长率** | 总资产周转率 | 进行历史和同业对比 | 营运能力分析，营业收入÷平均总资产 |

## 四、 “利润表”及相关附注

| 项目 | 关注点 | 具体审查内容 | 处理方法 | 备注 |
| :--- | :--- | :--- | :--- | :--- |
| **营业收入** | 主营业务 | 公司主营业务涉猎特别广泛，财报“主营业务（行业分类）”表格较长，先对其价值打个折 | 警惕盲目多元化或以主业无发展潜力 | |
| **营业收入** | 确认原则 | 第一次阅读某公司财报，一定要了解收入确认准则。完工百分比法的操作空间比较大 | | |
| **营业收入** | 增长率 | 看增速是否高于行业平均水平，高于平均，证明市场份额在扩大，反之，说明市场份额在缩小 | 成长性分析，注意防止过于乐观 | |
| **营业收入** | 增长来源 | 在不考虑兼并式增长的情况下，收入增长的三种途径及关注点：潜在需求增长、市场份额扩大、价格提升 | 成长性分析，注意防止过于乐观 | 对于“佛龛市场”里的公司，须关注营业收入的增长，需要重点关注经营所得现金的去向和利用效率 |
| **营业收入** | 大客户依赖度 | 前五大客户营收占比高低，表明了公司的对外依赖程度 | | |
| **营业收入** | 可疑交易 | 1）公司销售以往没有生产或经营的产品；2）与某个以往财报中不存在的大客户交易，金额较大，交易方式显示双方具有一定的关联性；3）变更的供应商对于该客户而言，不具有合理用途，或价格明显偏离正常市场价；4）主要供应商或主要客户存在《执照没有年审、注册地在居民楼、虚假注册地、税务资料缺失等》 | 警惕虚构收入 | |
| **关联交易** | 异常情况 | 包括：交易条件价值和是否显著高于市场正常情况，公司是否生产大额关联交易，其是否有合理的关键原因；以及这些交易是否有必要性 | 根据对行业的了解和公司历史情况进行判断 | |
| **毛利率** | 同业比较 | 毛利率显著高于或低于同行，要么是外行，要么是骗子，需根据行业状况进行分析 | 盈利能力分析 | |
| **毛利率** | 历史比较 | 毛利率的提高，是成本下降，要么是售价提升。 | | |
| **毛利率** | 杜邦分析 | 根据净利率高低、效率资产周转快慢、杠杆系数大小这三个因素的值，将公司归入“茅台模式”、“沃尔玛模式”、“银行模式”之一 | | |
| **费用** | 销售费用、管理费用 | 管理费用增加额应当不超过营收增幅 | 经营风险分析 | |
| **费用** | 销售费用、管理费用 | 将两者的和占营业收入的比例与同业比较，某年同比大幅上升，且主要用于对销售人员的激励，需警惕公司是否遇到经营困难 | | |
| **费用** | 费用率 | 费用率=三费÷营业总收入，保守起见，如果财务费用是正数，就加进去，如果财务费用是负数，就减掉后再计算 | 警惕费用成本过高或剧烈变化的公司 | |
| **费用** | 费用占毛利润 | 如果经营成本为50%以内，算是优秀的公司，如果在30%-70%区域，但仍是具有一定优势的公司，如果超过70%，通常关注的价值不大 | | |
| **费用** | 研发费用 | 如果公司研发费用比较大，需要真的理解行业和公司后，才能投资 | 建议保守投资者放弃研发费用占比高的公司，除非你懂技术 | |
| **减值损失** | 常见操纵手段 | 不提减值，增加当期利润 | 警惕 | |
| **减值损失** | 常见操纵手段 | 超额计提，降低当期利润，甚至造成亏损，然后下期再按照公允价值出售，增加利润及利润的同比增速，例如：某年对某资产大额计提减值损失，第二年就将其出售，而且出售时的成交价显著高于账面成本 | 警惕 | |
| **减值损失** | 常见操纵手段 | 洗大澡，一次亏够，以便后期利润同比大增，例如：资产减值损失同比大增，同时发现如果按照上年利润水平计提，利润同比增幅不大 | 警惕 | |
| **减值损失** | 常见操纵手段 | 超额计提，降低资产账面价值，减少利润折旧和摊销，增加后期财报利润，例如：固定资产、存货、应收款项、金融资产等大幅计提减值损失 | 警惕 | |
| **营业利润** | 增长速度 | 营业利润增速不低于营业收入增速 | 成长性分析 | |
| **营业利润** | 营业利润率 | 进行历史对比和同业对比，一般营业利润率上升的主要原因为：售价提升、成本下降或费用控制 | 盈利能力分析 | |
| **其他业务收入** | 突然增加 | 其他业务收入占营业收入的比重突然出现大幅提升，需警惕 | | |
| **营业外收入** | 数值过大 | 如果营业外收入净额比较大，建议直接用营业利润的75%当作净利润 | | |
| **营业外收入** | 占比较大 | 公司利润的主要部分经常来自营业外收入，造假概率大 | 报警并远离 | |
| **营业外支出、资产减值损失** | 不知所云的费用科目 | 财报上出现金额较大的“其他XX”或一些很少见的科目名称 | 警惕公司隐藏问题 | |
| **净利润** | 含金量 | 用“经营现金流净额÷净利润”来衡量净利润的含金量 | 该值连续数年大于1，发现“印钞机”的概率大，反之，连续数年小于1，需警惕并分析原因 | |
| **净资产收益率(ROE)** | 数值 | 净利润÷平均净资产，越大越好，连续数年大于15%为优秀公司，进行历史对比和同业对比 | 盈利能力分析 | |
| **总资产收益率** | 数值 | 净利润÷平均总资产，在ROE的基础上考虑了杠杆因素，进行历史对比和同业对比 | 盈利能力分析 | |
| **资产结构** | 轻重资产 | 生产资产（主要包括固定资产，在建工程、工程物资及无形资产中的土地）占总资产的比例，占比大称为“重资产公司”，占比小称为“轻资产公司”。用“税前利润总额÷生产资产”，得出的比值如果显著高于社会平均资本回报率（银行借款标准利率的两倍左右），则属于优秀公司，是其竞争力的体现。 | 只是描述重资产公司，若遭遇市场不景气、价格战，容易不景气，除非有很强大的护城河，否则不建议在其行业高增长期之外的时间持有。 | |

## 五、 “现金流量表”及相关附注

| 项目 | 关注点 | 具体审查内容 | 处理方法 |
| :--- | :--- | :--- | :--- |
| **经营活动现金流** | 评估 | 若公司经营活动现金净额连续数年大于净利润，且大于折旧摊销，则公司很优秀 | |
| **经营活动现金流** | 评估 | 经营活动现金净额持续优于同行，说明公司在产业链上地位强，或管理优秀 | |
| **经营活动现金流** | 异常情况 | 经营活动现金净额为正，主要是因为应付账款和应付票据大量增加 | 警惕公司遇到经营困难 |
| **经营活动现金流** | 与利润对比 | 经营现金流净额持续多年为负，却同时有巨额利润 | 报警，公司利润可能是虚构的 |
| **投资活动现金流** | 资产情况 | 剔除理财产品因素后，投资活动现金流净额为负，且主要用于扩张，而非维持原有盈利能力，为优秀企业的特征 | |
| **投资活动现金流** | 投资回报 | 投资回报率通常应高于社会资金平均回报水平（可简单以贷款利率的两倍为参考） | |
| **投资活动现金流** | 异常情况 | 投资活动现金流入中的大量出售为出售固定资产或其他长期资产 | 可能是经营不善，靠变卖家产度日 |
| **投资活动现金流** | 异常情况 | 投资活动现金流出金额超过年初预算很多 | 警惕管理层乱花钱 |
| **筹资活动现金流** | 异常情况 | 公司明明需要大量现金，却偿还了大部分贷款，出现“取得借款收到的现金”远小于“偿还债务支付的现金” | 可能是银行降低了贷款意愿 |
| **筹资活动现金流** | 异常情况 | 借款利率显著高于或低于市场常见利率（体现在“分配股利、利润和偿付利息的现金”和“支付其他与筹资活动有关的现金”等科目的附注里） | 可能遇到生存危机 |
| **筹资活动现金流** | 异常情况 | 公司一边大量借款，一边却将大笔资金用于理财，且理财收益率显著低于借款利率 | 危险和警惕指标 |
| **现金及现金等价物净增加额** | 三大活动的现金增加额 | 大于0，则公司基本健康。可放宽为：排除当年实施的现金分红因素影响后，大于0。 | |
| **现金流量表** | 值得关注的类型 | “正负负”型，即经营现金流入，投资和筹资现金流出。关注：投资现金流出是否用于扩大生产而非维持现有盈利规模，筹资现金流出是否用于分红而非还债。 | |
| **现金流量表** | 值得关注的类型 | “正正负”型，即经营和投资现金流入，筹资现金流出。关注：根据对公司和行业的理解思考投资现金流入的来源是否可持续，同时，警惕投资现金流入是否来自出售核心资产。 | |
