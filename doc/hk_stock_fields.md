# 港股财务数据字段清单

## 📋 概述

基于 FastAPI 字段发现 API 的完整 港股财务数据字段清单，包含财务指标、资产负债表、利润表和现金流量表的所有可用字段。

## 🔍 字段发现API端点

- **财务指标**: `GET /api/v1/financial/fields/hk_stock/hk_stock_indicators` (37个字段)
- **资产负债表**: `GET /api/v1/financial/fields/hk_stock/hk_stock_balance_sheet` (73个字段)
- **利润表**: `GET /api/v1/financial/fields/hk_stock/hk_stock_income_statement` (35个字段)
- **现金流量表**: `GET /api/v1/financial/fields/hk_stock/hk_stock_cash_flow` (64个字段)

---

## 🔧 使用说明

### 1. API 调用方式

```python
import httpx

# 获取所有可用字段
response = httpx.get("http://localhost:8000/api/v1/financial/fields/hk_stock/hk_stock_balance_sheet")
fields = response.json()["data"]["columns"]  # 注意：字段在 "columns" 中，不是 "metadata"
```

### 2. 查询参数

- **market**: `hk_stock` (港股市场)
- **query_type**:
  - `hk_stock_indicators` (财务指标)
  - `hk_stock_balance_sheet` (资产负债表)
  - `hk_stock_income_statement` (利润表)
  - `hk_stock_cash_flow` (现金流量表)

### 3. 字段选择建议

根据财务分析需求选择相关字段：
- **基础分析**: 选择核心指标字段
- **详细分析**: 选择完整的报表字段
- **特定检查**: 根据检查清单需求选择特定字段组合
