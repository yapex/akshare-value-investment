# 港股财务数据字段清单

## 📋 概述

基于 FastAPI 字段发现 API 的完整港股财务数据字段清单，包含财务指标、资产负债表、利润表和现金流量表的所有可用字段。

## 🔍 字段发现API端点

- **财务指标**: `GET /api/v1/financial/fields/hk_stock/hk_stock_indicators` (37个字段)
- **财务三表**: `GET /api/v1/financial/fields/hk_stock/hk_financial_statements`
  - 资产负债表: 73个字段
  - 利润表: 35个字段
  - 现金流量表: 64个字段

## 🔧 使用说明

### 1. API 调用方式

```python
import httpx

# 获取财务指标字段
response = httpx.get("http://localhost:8000/api/v1/financial/fields/hk_stock/hk_stock_indicators")
indicators_fields = response.json()["data"]["columns"]

# 获取财务三表字段
response = httpx.get("http://localhost:8000/api/v1/financial/fields/hk_stock/hk_financial_statements")
data = response.json()["data"]

# 资产负债表字段
balance_fields = data["balance_sheet"]["columns"]

# 利润表字段
income_fields = data["income_statement"]["columns"]

# 现金流量表字段
cashflow_fields = data["cash_flow"]["columns"]
```

### 2. 查询参数

- **market**: `hk_stock` (港股市场)
- **query_type**:
  - `hk_stock_indicators` (财务指标)
  - `hk_financial_statements` (财务三表聚合)
