# 📊 财报检查清单
## 一、"资产负债表"及相关附注

### 1.1 资产类项目

#### 1.1.1 货币资金是否安全？
- **状态：** ✅ **可量化**
- **追问：** 现金及现金等价物能否覆盖有息负债？若不能，加上可迅速变现的金融资产后能否覆盖？
- **计算公式：**
  ```
  货币资金安全比率 = (`货币资金` + `交易性金融资产`) ÷ (`短期借款` + `长期借款`) × 100%
  总覆盖率 = (`货币资金` + `交易性金融资产` + `总现金`) ÷ (`短期借款` + `长期借款`) × 100%
  ```
- **可用字段：**
  - 资产负债表：`货币资金`, `交易性金融资产`, `总现金`, `短期借款`, `长期借款`

#### 1.1.2 货币资金是否存在异常？
- **状态：** ✅ **可量化**
- **追问：** 余额是否远小于短期负债？是否存在大额受限资金？利息收入是否显著低于市场利率？
- **计算公式：**
  ```
  资金覆盖度 = `货币资金` ÷ `短期借款` × 100%
  估算利率 = `利息收入` ÷ `货币资金` × 100%
  资金负债比 = `货币资金` ÷ (`短期借款` + `长期借款`) × 100%
  ```
- **可用字段：**
  - 资产负债表：`货币资金`, `短期借款`, `长期借款`
  - 利润表：`利息收入`

#### 1.1.3 应收票据是否健康？
- **状态：** ❌ **无法量化**
- **追问：** 商业承兑汇票占比是否过高？是否突然大量出现商票？应收票据规模是否与正常经营规模匹配？
- **说明：** 虽然有`应收票据及应收账款`和`其中：应收票据`字段，但缺少区分商业承兑汇票和银行承兑汇票的详细信息，无法判断商票占比是否过高。

#### 1.1.4 应收账款是否健康？
- **状态：** ✅ **可量化**
- **追问：**
  - 增长是否快于收入？
  - 占总资产或收入的比例是否过大？
  - 账龄是否过长（如超一年）？
  - 周转率是否呈下降趋势或低于行业水平？
  - 有无集中或异常的应收款？
- **计算公式：**
  ```
  应收账款占总资产比例 = (`应收账款` + `其他应收款`) ÷ `资产合计` × 100%
  应收账款周转率 = `营业收入` ÷ (`应收账款` + `其他应收款`) × 2  # 需要两期数据计算平均值
  ```
- **可用字段：**
  - 资产负债表：`应收账款`, `其他应收款`, `资产合计`
  - 利润表：`营业收入`

#### 1.1.5 预付账款是否异常？
- **状态：** ❌ **无法量化**
- **追问：** 是否大幅增长（尤其是工程款、技术采购款）？有无长期挂账？占收入或成本的比例是否大幅波动？
- **说明：** 虽然有`预付款项`字段，但缺少历史对比和行业对比数据，无法判断是否大幅增长或长期挂账。

#### 1.1.6 其他应收款是否异常？
- **状态：** ✅ **可量化**（部分量化）
- **追问：** 剔除应收股利和利息后，数额是否较大？
- **计算公式：**
  ```
  其他应收款占比 = `其他应收款` ÷ `资产合计` × 100%
  ```
- **可用字段：**
  - 资产负债表：`其他应收款`, `资产合计`
- **说明：** 无法区分应收股利和利息的具体数额，因为只有`其他应收款合计`和`其他应收款`字段

#### 1.1.7 坏账准备计提是否合理？
- **状态：** ✅ **可量化**（替代方案）
- **追问：** 计提标准与同行相比如何？有无变更标准？有无异常的统一比例计提或应提未提情况？
- **计算公式：**
  ```
  坏账准备比率 = (`资产减值损失` + `信用减值损失`) ÷ (`应收账款` + `其他应收款`) × 100%
  ```
- **可用字段：**
  - 资产负债表：`应收账款`, `其他应收款`
  - 利润表：`资产减值损失`, `信用减值损失`
- **说明：** 由于缺少直接的"坏账准备"字段，使用利润表中的"资产减值损失"和"信用减值损失"作为替代指标

#### 1.1.8 存货是否存在风险？
- **状态：** ✅ **可量化**
- **追问：**
  - 存货（特别是易贬值品）的减值准备计提是否充分？
  - 存货增长是否快于营业成本？
  - 存货周转率下降的同时毛利率是否异常提升？
- **计算公式：**
  ```
  存货周转率 = `营业成本` ÷ `存货` × 2  # 需要两期数据计算平均值
  ```
- **可用字段：**
  - 资产负债表：`存货`
  - 利润表：`营业成本`

#### 1.1.9 公司是否属于财务造假高危领域？
- **状态：** ❌ **无法量化**
- **追问：** 是否为农林牧渔公司？税收优惠是否多？产品是否不易核查或估值？
- **说明：** 这是一个定性判断，需要行业分类和业务模式分析，无法通过财务数据直接量化。

#### 1.1.10 长期股权投资是否存在操纵迹象？
- **状态：** ❌ **无法量化**
- **追问：**
  - 有无通过小额交易改变股权分类以调节利润？
  - 交易对手方是否疑似同一控制？
  - 有无将一笔交易分拆多笔进行？
- **说明：** 需要具体交易细节和关联方信息，仅靠财务报表数据难以判断是否存在操纵迹象。

#### 1.1.11 投资性房地产的计量模式是什么？
- **状态：** ❌ **无法量化**
- **追问：** 是以成本模式还是公允价值模式计量？
- **说明：** 会计政策信息在报表附注中说明，无法通过数值字段直接判断计量模式。

#### 1.1.12 固定资产折旧政策是否变更？
- **状态：** ❌ **无法量化**
- **追问：** 变更原因及影响是否合理？有无账面价值极低但实际价值未降的资产（如酒窖、房产）？
- **计算公式：**
  ```
  固定资产周转率 = `营业收入` ÷ `固定资产合计` × 100%  # 只能使用账面现值近似
  ```
- **可用字段：** `固定资产合计`, `营业收入`
- **说明：** 会计政策变更信息在报表附注中，且字段中缺少固定资产净值（原值-累计折旧）相关信息，无法准确计算周转率。

#### 1.1.13 在建工程是否异常？
- **状态：** ✅ **可量化**（部分量化）
- **追问：**
  - 有无已完工但迟迟不转固？
  - 有无不断超预算追加投资？
  - 工程进度与资金投入是否匹配？
  - 转固后是否很快需要大修？
- **计算公式：**
  ```
  在建工程占比 = `在建工程合计` ÷ `资产合计` × 100%
  ```
- **可用字段：**
  - 资产负债表：`在建工程合计`, `资产合计`
- **说明：** 无法判断完工状态和转固时间，需要对比多期数据。

#### 1.1.14 开发支出资本化是否合理？
- **状态：** ❌ **无法量化**
- **追问：** 需结合行业与公司特点判断资本化政策的合理性。
- **说明：** "开发支出"字段未在资产负债表字段清单中找到，可能归类在"其他非流动资产"中，无法直接量化评估其合理性。

#### 1.1.15 商誉是否过大？
- **状态：** ❌ **无法量化**
- **追问：**
  - 收购作价是否公允？
  - 大额商誉是否使负债率堪忧？
  - 减值测试披露是否充分（有无过程披露）？
  - 收购并出售大额商誉资产的交易是否可疑？
- **计算公式：**
  ```
  调整后负债率 = (`负债合计` + 商誉) ÷ `资产合计` × 100%  # 商誉字段缺失
  ```
- **说明：** "商誉"字段未在资产负债表字段清单中找到，可能包含在"其他非流动资产"中，无法直接计算调整后负债率。

#### 1.1.16 长期待摊费用数额是否过大？
- **状态：** ✅ **可量化**
- **追问：** 是否可能存在不恰当的费用资本化？
- **计算公式：**
  ```
  长期待摊费用占比 = `长期待摊费用` ÷ `资产合计` × 100%
  ```
- **可用字段：** `长期待摊费用`, `资产合计`

### 1.2 负债与权益类项目

#### 1.2.1 有息负债占比及成本如何？
- **状态：** ✅ **可量化**
- **追问：**
  - 有息负债占总资产比例与行业龙头相比如何？
  - 借款利率是否显著高于市场水平？
  - 是否在货币资金充裕时仍高息借款？
- **计算公式：**
  ```
  有息负债占比 = (`短期借款` + `长期借款`) ÷ `资产合计` × 100%
  资产负债率 = `负债合计` ÷ `资产合计` × 100%
  借款利率 = `财务费用` ÷ (`短期借款` + `长期借款`) × 100%  # 需要平均值，但可用当期值近似
  ```
- **可用字段：**
  - 资产负债表：`短期借款`, `长期借款`, `资产合计`, `负债合计`
  - 利润表：`财务费用`

#### 1.2.2 应付账款/票据是否异常大增？
- **状态：** ✅ **可量化**（部分量化）
- **追问：** 是行业地位提升所致，还是资金链紧张的表现？
- **计算公式：**
  ```
  应付账款占比 = `应付账款` ÷ `负债合计` × 100%
  应付票据及账款占比 = `应付票据及应付账款` ÷ `负债合计` × 100%
  ```
- **可用字段：** `应付账款`, `应付票据及应付账款`, `负债合计`
- **说明：** 需要多期数据对比才能判断是否大增

#### 1.2.3 预收款项数额是否略高？
- **状态：** ✅ **可量化**（部分量化）
- **追问：** 是否存在推迟确认收入、隐藏利润的可能？
- **计算公式：**
  ```
  预收账款占比 = `预收款项` ÷ `负债合计` × 100%
  ```
- **可用字段：** `预收款项`, `负债合计`
- **说明：** 需要行业对比和历史数据判断是否过高

#### 1.2.4 应付职工薪酬是否透露关键信息？
- **状态：** ❌ **无法量化**
- **追问：**
  - 高管及核心人员薪酬与竞争力是否匹配？
  - 员工平均薪酬是否合理（历史与同业对比）？
  - 公司盈利对工资上调的敏感性如何（工资黏性）？
  - 有无通过预提薪酬平滑利润的迹象（结合递延所得税资产看）？
- **计算公式：**
  ```
  员工薪酬占比 = `应付职工薪酬` ÷ `营业收入` × 100%
  ```
- **可用字段：** `应付职工薪酬`, `营业收入`
- **说明：** 虽然有`应付职工薪酬`字段，但缺少员工总数和薪酬历史数据，无法进行详细分析。

#### 1.2.5 有无隐藏的良性债务？
- **状态：** ✅ **可量化**
- **追问：** 专项应付款、递延收益余额是否较大？
- **计算公式：**
  ```
  专项应付款占比 = `专项应付款` ÷ `负债合计` × 100%
  ```
- **可用字段：** `专项应付款`, `负债合计`

#### 1.2.6 库存股回购的目的是什么？
- **状态：** ✅ **可量化**
- **追问：** 是用于注销（价格是否合理）还是股权激励（费用分摊如何）？
- **计算公式：**
  ```
  库存股占比 = `减：库存股` ÷ `资产合计` × 100%
  ```
- **可用字段：** `减：库存股`, `资产合计`

### 1.3 整体分析指标

#### 1.3.1 公司成长性与营运能力如何？
- **状态：** ✅ **可量化**
- **追问：** 总资产增长率、净资产增长率、总资产周转率与行业龙头及自身历史相比如何？
- **计算公式：**
  ```
  总资产周转率 = `营业收入` ÷ `资产合计` × 2  # 需要期初期末平均值
  ```
- **可用字段：** `资产合计`, `营业收入`
- **说明：** 需要两期数据来计算增长率

## 二、"利润表"及相关附注

### 2.1 主营业务是否聚焦？
- **状态：** ❌ **无法量化**
- **追问：** 主营业务是否过于多元化？收入确认原则（特别是完工百分比法）是否清晰？
- **说明：** 需要业务分部信息和收入构成明细，现有利润表字段中只有总体收入类别，无法判断主营业务是否聚焦。

### 2.2 营业收入增长是否健康？
- **状态：** ✅ **可量化**
- **追问：**
  - 增速是否高于行业平均？
  - 增长来源于需求、份额还是提价？
  - 对大客户依赖度是否过高？
- **计算公式：**
  ```
  营业收入增长率 = (`营业收入` - 上期营业收入) ÷ 上期营业收入 × 100%
  ```
- **可用字段：** `营业收入`
- **说明：** 需要多期数据对比来计算增长率

### 2.3 有无可疑的收入或关联交易？
- **状态：** ❌ **无法量化**
- **追问：**
  - 是否销售陌生产品？
  - 是否突现大额陌生客户交易？
  - 关联交易的必要性、定价是否合理？
- **说明：** 关联交易信息在报表附注中详细披露，仅凭利润表数据无法判断交易的可疑性。

### 2.4 毛利率是否异常？
- **状态：** ✅ **可量化**
- **追问：** 与同行相比是否显著偏离？历史变动原因是成本还是售价？
- **计算公式：**
  ```
  毛利率 = (`营业收入` - `营业成本`) ÷ `营业收入` × 100%
  ```
- **可用字段：** `营业收入`, `营业成本`

### 2.5 费用控制是否有效？
- **状态：** ✅ **可量化**
- **追问：**
  - 管理费用增幅是否超过营收？
  - 销售与管理费用率与同行相比如何？
  - 三费（费用率）占毛利润的比例是多少？
  - 研发费用占比是否过高（若不懂则谨慎）？
- **计算公式：**
  ```
  销售费用率 = `销售费用` ÷ `营业收入` × 100%
  管理费用率 = `管理费用` ÷ `营业收入` × 100%
  财务费用率 = `财务费用` ÷ `营业收入` × 100%
  三费占营收比 = (`销售费用` + `管理费用` + `财务费用`) ÷ `营业收入` × 100%
  研发费用占比 = `研发费用` ÷ `营业收入` × 100%
  ```
- **可用字段：** `营业收入`, `营业成本`, `销售费用`, `管理费用`, `财务费用`, `研发费用`

### 2.6 资产减值损失有无操纵痕迹？
- **状态：** ✅ **可量化**（部分量化）
- **追问：** 有无不提、超额计提、洗大澡等迹象？
- **计算公式：**
  ```
  资产减值影响 = (`资产减值损失` + `信用减值损失`) ÷ `营业收入` × 100%
  ```
- **可用字段：** `资产减值损失`, `信用减值损失`, `营业收入`
- **说明：** 需要多期数据对比才能判断是否存在操纵痕迹

### 2.7 营业利润增长质量如何？
- **状态：** ✅ **可量化**
- **追问：** 增速是否不低于营收增速？营业利润率变动原因是什么？
- **计算公式：**
  ```
  营业利润增长率 = (`营业利润` - 上期营业利润) ÷ 上期营业利润 × 100%
  营业收入增长率 = (`营业收入` - 上期营业收入) ÷ 上期营业收入 × 100%
  营业利润率 = `营业利润` ÷ `营业收入` × 100%
  ```
- **可用字段：** `营业利润`, `营业收入`
- **说明：** 需要多期数据对比

### 2.8 其他收益是否可靠？
- **状态：** ✅ **可量化**（部分量化）
- **追问：**
  - 其他业务收入占比是否突然大增？
  - 营业外收入净额是否过大？
  - 利润是否主要依赖营业外收入？
- **计算公式：**
  ```
  营业外净收入 = `营业外收入` - `营业外支出`
  营业外收入占比 = `营业外收入` ÷ `净利润` × 100%
  其他收益占比 = `其他收益` ÷ `营业收入` × 100%
  ```
- **可用字段：** `营业外收入`, `营业外支出`, `其他收益`, `净利润`, `营业收入`
- **说明：** "其他业务收入"字段不存在，无法计算其占比

### 2.9 净利润含金量高吗？
- **状态：** ✅ **可量化**
- **追问：** 经营现金流净额连续数年是否大于净利润？
- **计算公式：**
  ```
  净利润含金量 = `经营活动产生的现金流量净额` ÷ `净利润` × 100%
  ```
- **可用字段：** `净利润`, `经营活动产生的现金流量净额`

### 2.10 核心收益率指标如何？
- **状态：** ✅ **可量化**
- **追问：** 净资产收益率(ROE)是否连续大于15%？总资产收益率(ROA)历史与同业对比如何？
- **计算公式：**
  ```
  净资产收益率(ROE) = `净利润` ÷ `归属于母公司所有者权益合计` × 100%  # 需要平均值
  总资产收益率(ROA) = `净利润` ÷ `资产合计` × 100%  # 需要平均值
  ```
- **可用字段：** `净利润`, `归属于母公司所有者权益合计`, `资产合计`
- **说明：** 财务指标表中直接提供 `净资产收益率` 和 `净资产收益率-摊薄` 字段

### 2.11 公司属于重资产还是轻资产？
- **状态：** ✅ **可量化**（部分量化）
- **追问：** 生产资产占比如何？其税前回报率是否显著高于社会平均资本回报率？
- **计算公式：**
  ```
  生产资产占比 = (`固定资产合计` + `在建工程合计` + `工程物资` + `无形资产`) ÷ `资产合计` × 100%
  生产资产回报率 = `利润总额` ÷ (`固定资产合计` + `在建工程合计` + `工程物资` + `无形资产`) × 100%  # 税前利润用利润总额替代
  ```
- **可用字段：** `固定资产合计`, `在建工程合计`, `工程物资`, `无形资产`, `资产合计`, `利润总额`
- **说明：** "税前利润总额"字段不存在，使用"利润总额"替代

## 三、"现金流量表"及相关附注

### 3.1 经营活动现金流是否优秀？
- **状态：** ✅ **可量化**
- **追问：**
  - 是否连续数年大于净利润及折旧摊销？
  - 是否持续优于同行？
  - 若为正，是否主要靠应付项目增加？
  - 若持续为负却有巨额利润，是否报警？
- **计算公式：**
  ```
  经营现金流健康度 = `经营活动产生的现金流量净额` ÷ `净利润` × 100%
  经营现金流与净利润比 = `经营活动产生的现金流量净额` ÷ (`净利润` + `固定资产折旧、油气资产折耗、生产性生物资产折旧` + `无形资产摊销` + `长期待摊费用摊销`)
  ```
- **可用字段：** `经营活动产生的现金流量净额`, `净利润`, `固定资产折旧、油气资产折耗、生产性生物资产折旧`, `无形资产摊销`, `长期待摊费用摊销`

### 3.2 投资活动现金流是否健康？
- **状态：** ✅ **可量化**
- **追问：**
  - 剔除理财后，净流出是否主要用于扩张？
  - 投资回报率是否高于资金平均成本？
  - 现金流入是否大量来自变卖资产？
  - 流出是否严重超预算？
- **计算公式：**
  ```
  投资活动净现金流 = `投资活动产生的现金流量净额`
  投资回报率 = (`投资收益` + `取得投资收益收到的现金`) ÷ |`投资活动产生的现金流量净额`| × 100%
  ```
- **可用字段：** `投资活动产生的现金流量净额`, `投资收益`, `取得投资收益收到的现金`
- **说明：** 需要区分理财投资与实业投资，但数据中未明确分类

### 3.3 筹资活动现金流是否异常？
- **状态：** ✅ **可量化**（部分量化）
- **追问：**
  - 在需要现金时是否大量还贷（银行缩贷）？
  - 借款利率是否显著偏离市场？
  - 是否同时存在高息借款和低收益理财？
- **计算公式：**
  ```
  筹资活动净现金流 = `筹资活动产生的现金流量净额`
  借款利率 = `分配股利、利润或偿付利息支付的现金` ÷ (`短期借款` + `长期借款`) × 100%
  ```
- **可用字段：** `筹资活动产生的现金流量净额`, `分配股利、利润或偿付利息支付的现金`, `短期借款`, `长期借款`
- **说明：** 无法直接判断是否同时存在高息借款和低收益理财

### 3.4 公司整体现金流类型是什么？
- **状态：** ✅ **可量化**
- **追问：**
  - 是否为"正负负"（经营流入，投资、筹资流出）或"正正负"（经营、投资流入，筹资流出）型？
  - 投资流出是用于扩张还是维持？
  - 筹资流出是用于分红还是还债？
  - 投资流入是否可持续、是否变卖核心资产？
- **分类标准：**
  - **"正负负"型**：`经营活动产生的现金流量净额` > 0 且 `投资活动产生的现金流量净额` < 0 且 `筹资活动产生的现金流量净额` < 0
  - **"正正负"型**：`经营活动产生的现金流量净额` > 0 且 `投资活动产生的现金流量净额` > 0 且 `筹资活动产生的现金流量净额` < 0
- **可用字段：**
  - `经营活动产生的现金流量净额`
  - `投资活动产生的现金流量净额`
  - `筹资活动产生的现金流量净额`

## 📊 财务字段映射表

### 🏦 三市场财务字段映射说明

财报检查清单主要基于 A 市场编写，但可以通过字段映射应用于 A 股、港股、美股三个市场。


#### 📋 字段映射原则

1. **功能性映射**: 按财务功能（资产、负债、利润等）进行分类
2. **市场适配**: 每个市场提供对应的字段名称
3. **查询方式**: 使用 FastAPI 的 `/financial/query` 查询数据
4. **可用字段验证**: 通过以下 API 端点获取可用字段列表
   - **财务指标**: `GET /api/v1/financial/fields/a_stock/a_stock_indicators` (25个字段)
   - **资产负债表**: `GET /api/v1/financial/fields/a_stock/a_stock_balance_sheet` (75个字段)
   - **利润表**: `GET /api/v1/financial/fields/a_stock/a_stock_income_statement` (46个字段)
   - **现金流量表**: `GET /api/v1/financial/fields/a_stock/a_stock_cash_flow` (72个字段)

#### 📊 字段可用性统计

- **可量化项目：19个** (约占61%)
- **无法量化项目：12个** (约占39%)

##### 主要字段缺失
- **商誉**：在字段清单中未找到，可能包含在"其他非流动资产"中
- **开发支出**：未在字段清单中找到
- **累计折旧**：无法直接获得固定资产净值
- **其他业务收入**：与主营业务收入合并为"营业收入"

##### 替代方案
- **坏账准备** → 使用"资产减值损失"+"信用减值损失"
- **存货跌价准备** → 使用"资产减值损失"
- **净资产收益率** → 财务指标表直接提供
- **税前利润** → 使用"利润总额"
