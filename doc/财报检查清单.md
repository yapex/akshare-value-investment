# 📊 财报检查清单
## 一、"资产负债表"及相关附注

### 1.1 资产类项目

#### 1.1.1 货币资金是否安全？
- **状态：** ✅ **可量化**
- **追问：** 现金及现金等价物能否覆盖有息负债？若不能，加上可迅速变现的金融资产后能否覆盖？
- **计算公式：**
  ```
  货币资金安全比率 = (`货币资金` + `交易性金融资产`) ÷ (`短期借款` + `长期借款`) × 100%
  总覆盖率 = (`货币资金` + `交易性金融资产` + `总现金`) ÷ (`短期借款` + `长期借款`) × 100%
  ```
- **三地市场字段映射：**

  | 项目 | A股 | 港股 | 美股 |
  |------|-----|------|------|
  | 货币资金 | `货币资金` | `现金及等价物` | `现金及现金等价物` |
  | 交易性金融资产 | `交易性金融资产` | `交易性金融资产(流动)` | `有价证券投资(流动)` |
  | 短期借款 | `短期借款` | `短期贷款` | `短期债务` |
  | 长期借款 | `长期借款` | `长期贷款` | `长期负债` |

- **安全标准：**
  - 优秀：安全比率 ≥ 100%（现金完全覆盖有息负债）
  - 良好：加上金融资产后 ≥ 100%
  - 风险：即使加上金融资产仍 < 100%

#### 1.1.2 货币资金是否存在异常？
- **状态：** ✅ **可量化**
- **追问：** 余额是否远小于短期负债？是否存在大额受限资金？利息收入是否显著低于市场利率？
- **计算公式：**
  ```
  资金覆盖度 = `货币资金` ÷ `短期借款` × 100%
  估算利率 = `利息收入` ÷ `货币资金` × 100%
  资金负债比 = `货币资金` ÷ (`短期借款` + `长期借款`) × 100%
  ```
- **三地市场字段映射：**

  | 项目 | A股 | 港股 | 美股 |
  |------|-----|------|------|
  | 货币资金 | `货币资金` | `现金及等价物` | `现金及现金等价物` |
  | 短期借款 | `短期借款` | `短期贷款` | `短期债务` |
  | 长期借款 | `长期借款` | `长期贷款` | `长期负债` |
  | 利息收入 | `利息收入` | `利息收入` | `利息收入` |

- **异常判断标准：**
  - 资金覆盖度 < 50%：异常
  - 估算利率 < 市场利率50%：可能存在受限资金
  - 资金负债比 < 30%：高度风险

#### 1.1.3 应收票据是否健康？
- **状态：** ❌ **无法量化**
- **追问：** 商业承兑汇票占比是否过高？是否突然大量出现商票？应收票据规模是否与正常经营规模匹配？
- **说明：** 虽然有`应收票据及应收账款`和`其中：应收票据`字段，但缺少区分商业承兑汇票和银行承兑汇票的详细信息，无法判断商票占比是否过高。

#### 1.1.4 应收账款是否健康？
- **状态：** ✅ **可量化**
- **追问：**
  - 增长是否快于收入？
  - 占总资产或收入的比例是否过大？
  - 账龄是否过长（如超一年）？
  - 周转率是否呈下降趋势或低于行业水平？
  - 有无集中或异常的应收款？
- **计算公式：**
  ```
  应收账款占总资产比例 = (`应收账款` + `其他应收款`) ÷ `资产合计` × 100%
  应收账款周转率 = `营业收入` ÷ (`应收账款` + `其他应收款`) × 2  # 需要两期数据计算平均值
  ```
- **三地市场字段映射：**

  | 项目 | A股 | 港股 | 美股 |
  |------|-----|------|------|
  | 应收账款 | `应收账款` | `应收帐款` | `应收账款` |
  | 其他应收款 | `其他应收款` | `其他金融资产(流动)` | `其他应收款` |
  | 资产合计 | `资产合计` | `总资产` | `总资产` |
  | 营业收入 | `营业收入` | `营业额` | `营业收入` |

- **健康标准：**
  - 占总资产比例 < 20%：健康
  - 周转率 > 行业平均：健康
  - 增长率 < 营收增长率：健康

#### 1.1.5 预付账款是否异常？
- **状态：** ❌ **无法量化**
- **追问：** 是否大幅增长（尤其是工程款、技术采购款）？有无长期挂账？占收入或成本的比例是否大幅波动？
- **说明：** 虽然有`预付款项`字段，但缺少历史对比和行业对比数据，无法判断是否大幅增长或长期挂账。

#### 1.1.6 其他应收款是否异常？
- **状态：** ✅ **可量化**（部分量化）
- **追问：** 剔除应收股利和利息后，数额是否较大？
- **计算公式：**
  ```
  其他应收款占比 = `其他应收款` ÷ `资产合计` × 100%
  ```
- **三地市场字段映射：**

  | 项目 | A股 | 港股 | 美股 |
  |------|-----|------|------|
  | 其他应收款 | `其他应收款` | `其他金融资产(流动)` | `其他应收款` |
  | 资产合计 | `资产合计` | `总资产` | `总资产` |

- **异常标准：**
  - 占比 > 5%：需要关注
  - 占比 > 10%：异常，可能存在关联方占用资金
- **说明：** 无法区分应收股利和利息的具体数额，因为只有`其他应收款合计`和`其他应收款`字段

#### 1.1.7 坏账准备计提是否合理？
- **状态：** ✅ **可量化**（替代方案）
- **追问：** 计提标准与同行相比如何？有无变更标准？有无异常的统一比例计提或应提未提情况？
- **计算公式：**
  ```
  坏账准备比率 = (`资产减值损失` + `信用减值损失`) ÷ (`应收账款` + `其他应收款`) × 100%
  ```
- **三地市场字段映射：**

  | 项目 | A股 | 港股 | 美股 |
  |------|-----|------|------|
  | 资产减值损失 | `资产减值损失` | `加:减值及拨备` | `减值及拨备` |
  | 信用减值损失 | `信用减值损失` | 不适用 | 不适用 |
  | 应收账款 | `应收账款` | `应收帐款` | `应收账款` |
  | 其他应收款 | `其他应收款` | `其他金融资产(流动)` | `其他应收款` |

- **合理性标准：**
  - 比率 1%-5%：正常
  - 比率 < 1%：可能计提不足
  - 比率 > 10%：可能存在大额坏账
- **说明：** 由于缺少直接的"坏账准备"字段，使用利润表中的"资产减值损失"和"信用减值损失"作为替代指标

#### 1.1.8 存货是否存在风险？
- **状态：** ✅ **可量化**
- **追问：**
  - 存货（特别是易贬值品）的减值准备计提是否充分？
  - 存货增长是否快于营业成本？
  - 存货周转率下降的同时毛利率是否异常提升？
- **计算公式：**
  ```
  存货周转率 = `营业成本` ÷ `存货` × 2  # 需要两期数据计算平均值
  存货占比 = `存货` ÷ `资产合计` × 100%
  ```
- **三地市场字段映射：**

  | 项目 | A股 | 港股 | 美股 |
  |------|-----|------|------|
  | 存货 | `存货` | `存货` | `存货` |
  | 营业成本 | `营业成本` | 不适用 | `主营成本` |
  | 资产合计 | `资产合计` | `总资产` | `总资产` |

- **风险标准：**
  - 周转率 > 行业平均：健康
  - 存货占比 < 20%：健康
  - 周转率连续下降：风险信号

#### 1.1.9 公司是否属于财务造假高危领域？
- **状态：** ❌ **无法量化**
- **追问：** 是否为农林牧渔公司？税收优惠是否多？产品是否不易核查或估值？
- **说明：** 这是一个定性判断，需要行业分类和业务模式分析，无法通过财务数据直接量化。

#### 1.1.10 长期股权投资是否存在操纵迹象？
- **状态：** ❌ **无法量化**
- **追问：**
  - 有无通过小额交易改变股权分类以调节利润？
  - 交易对手方是否疑似同一控制？
  - 有无将一笔交易分拆多笔进行？
- **说明：** 需要具体交易细节和关联方信息，仅靠财务报表数据难以判断是否存在操纵迹象。

#### 1.1.11 投资性房地产的计量模式是什么？
- **状态：** ❌ **无法量化**
- **追问：** 是以成本模式还是公允价值模式计量？
- **说明：** 会计政策信息在报表附注中说明，无法通过数值字段直接判断计量模式。

#### 1.1.12 固定资产折旧政策是否变更？
- **状态：** ❌ **无法量化**
- **追问：** 变更原因及影响是否合理？有无账面价值极低但实际价值未降的资产（如酒窖、房产）？
- **计算公式：**
  ```
  固定资产周转率 = `营业收入` ÷ `固定资产合计` × 100%  # 只能使用账面现值近似
  ```
- **可用字段：** `固定资产合计`, `营业收入`
- **说明：** 会计政策变更信息在报表附注中，且字段中缺少固定资产净值（原值-累计折旧）相关信息，无法准确计算周转率。

#### 1.1.13 在建工程是否异常？
- **状态：** ✅ **可量化**（部分量化）
- **追问：**
  - 有无已完工但迟迟不转固？
  - 有无不断超预算追加投资？
  - 工程进度与资金投入是否匹配？
  - 转固后是否很快需要大修？
- **计算公式：**
  ```
  在建工程占比 = `在建工程合计` ÷ `资产合计` × 100%
  固定资产在建比 = `在建工程合计` ÷ (`固定资产合计` + `在建工程合计`) × 100%
  ```
- **三地市场字段映射：**

  | 项目 | A股 | 港股 | 美股 |
  |------|-----|------|------|
  | 在建工程合计 | `在建工程合计` | `在建工程` | 不适用 |
  | 固定资产合计 | `固定资产合计` | `固定资产` | `物业、厂房及设备` |
  | 资产合计 | `资产合计` | `总资产` | `总资产` |

- **异常标准：**
  - 在建工程占比 > 10%：需要关注
  - 在建工程占比 > 20%：可能存在推迟转固
  - 固定资产在建比 > 30%：异常
- **说明：** 无法判断完工状态和转固时间，需要对比多期数据

#### 1.1.14 开发支出资本化是否合理？
- **状态：** ❌ **无法量化**
- **追问：** 需结合行业与公司特点判断资本化政策的合理性。
- **说明：** "开发支出"字段未在资产负债表字段清单中找到，可能归类在"其他非流动资产"中，无法直接量化评估其合理性。

#### 1.1.15 商誉是否过大？
- **状态：** ❌ **无法量化**
- **追问：**
  - 收购作价是否公允？
  - 大额商誉是否使负债率堪忧？
  - 减值测试披露是否充分（有无过程披露）？
  - 收购并出售大额商誉资产的交易是否可疑？
- **计算公式：**
  ```
  调整后负债率 = (`负债合计` + 商誉) ÷ `资产合计` × 100%  # 商誉字段缺失
  ```
- **说明：** "商誉"字段未在资产负债表字段清单中找到，可能包含在"其他非流动资产"中，无法直接计算调整后负债率。

#### 1.1.16 长期待摊费用数额是否过大？

- **追问：** 是否可能存在不恰当的费用资本化？


  |------|-----|------|------|
  | 长期待摊费用 | `长期待摊费用` | 不适用 | 不适用 |
  | 资产合计 | `资产合计` | `总资产` | `总资产` |

- **异常标准：**
  - 占比 < 1%：正常
  - 占比 1%-3%：需要关注
  - 占比 > 5%：可能存在费用资本化
- **说明：** 港股和美股无对应字段

### 1.2 负债与权益类项目

#### 1.2.1 有息负债占比及成本如何？
- **状态：** ✅ **可量化**
- **追问：**
  - 有息负债占总资产比例与行业龙头相比如何？
  - 借款利率是否显著高于市场水平？
  - 是否在货币资金充裕时仍高息借款？
- **计算公式：**
  ```
  有息负债占比 = (`短期借款` + `长期借款`) ÷ `资产合计` × 100%
  资产负债率 = `负债合计` ÷ `资产合计` × 100%
  借款利率 = `财务费用` ÷ (`短期借款` + `长期借款`) × 100%  # 需要平均值，但可用当期值近似
  ```
- **三地市场字段映射：**

  | 项目 | A股 | 港股 | 美股 |
  |------|-----|------|------|
  | 短期借款 | `短期借款` | `短期贷款` | `短期债务` |
  | 长期借款 | `长期借款` | `长期贷款` | `长期负债` |
  | 资产合计 | `资产合计` | `总资产` | `总资产` |
  | 负债合计 | `负债合计` | `总负债` | `总负债` |
  | 财务费用 | `财务费用` | `融资成本` | 不适用 |

- **健康标准：**
  - 有息负债占比 < 30%：健康
  - 资产负债率 40%-60%：合理
  - 借款利率 < 市场利率+2%：正常

#### 1.2.2 应付账款/票据是否异常大增？
- **状态：** ✅ **可量化**（部分量化）
- **追问：** 是行业地位提升所致，还是资金链紧张的表现？
- **计算公式：**
  ```
  应付账款占比 = `应付账款` ÷ `负债合计` × 100%
  应付票据及账款占比 = `应付票据及应付账款` ÷ `负债合计` × 100%
  ```
- **三地市场字段映射：**

  | 项目 | A股 | 港股 | 美股 |
  |------|-----|------|------|
  | 应付账款 | `应付账款` | `应付帐款` | `应付账款` |
  | 应付票据及应付账款 | `应付票据及应付账款` | 不适用 | 不适用 |
  | 负债合计 | `负债合计` | `总负债` | `总负债` |

- **判断标准：**
  - 占比 < 10%：正常
  - 占比 > 20%：可能资金链紧张
- **说明：** 需要多期数据对比才能判断是否大增

#### 1.2.3 预收款项数额是否略高？
- **状态：** ✅ **可量化**（部分量化）
- **追问：** 是否存在推迟确认收入、隐藏利润的可能？
- **计算公式：**
  ```
  预收账款占比 = `预收款项` ÷ `负债合计` × 100%
  ```
- **三地市场字段映射：**

  | 项目 | A股 | 港股 | 美股 |
  |------|-----|------|------|
  | 预收款项 | `预收款项` | 不适用 | `预收及预提费用` |
  | 负债合计 | `负债合计` | `总负债` | `总负债` |

- **判断标准：**
  - 占比 > 10%：需要关注（可能隐藏利润）
- **说明：** 需要行业对比和历史数据判断是否过高

#### 1.2.4 应付职工薪酬是否透露关键信息？
- **状态：** ❌ **无法量化**
- **追问：**
  - 高管及核心人员薪酬与竞争力是否匹配？
  - 员工平均薪酬是否合理（历史与同业对比）？
  - 公司盈利对工资上调的敏感性如何（工资黏性）？
  - 有无通过预提薪酬平滑利润的迹象（结合递延所得税资产看）？
- **计算公式：**
  ```
  员工薪酬占比 = `应付职工薪酬` ÷ `营业收入` × 100%
  ```
- **可用字段：** `应付职工薪酬`, `营业收入`
- **说明：** 虽然有`应付职工薪酬`字段，但缺少员工总数和薪酬历史数据，无法进行详细分析。

#### 1.2.5 有无隐藏的良性债务？
- **状态：** ✅ **可量化**
- **追问：** 专项应付款、递延收益余额是否较大？
- **计算公式：**
  ```
  专项应付款占比 = `专项应付款` ÷ `负债合计` × 100%
  ```
- **三地市场字段映射：**

  | 项目 | A股 | 港股 | 美股 |
  |------|-----|------|------|
  | 专项应付款 | `专项应付款` | 不适用 | 不适用 |
  | 负债合计 | `负债合计` | `总负债` | `总负债` |

- **良性债务特征：**
  - 无需偿还：政府补助、专项资金
  - 利息为零：降低财务成本
  - 占比 > 5%：显著优势

#### 1.2.6 库存股回购的目的是什么？
- **状态：** ✅ **可量化**
- **追问：** 是用于注销（价格是否合理）还是股权激励（费用分摊如何）？

- **异常标准：**
  - 筹资现金流为负（还贷或分红）：正常
  - 筹资现金流波动 > 100%：需关注
  - 借款利率 > 市场利率+3%：异常
- **说明：** 无法直接判断是否同时存在高息借款和低收益理财

### 3.4 公司整体现金流类型是什么？
- **状态：** ✅ **可量化**
- **追问：**
  - 是否为"正负负"（经营流入，投资、筹资流出）或"正正负"（经营、投资流入，筹资流出）型？
  - 投资流出是用于扩张还是维持？
  - 筹资流出是用于分红还是还债？
  - 投资流入是否可持续、是否变卖核心资产？
- **分类标准：**

  | 类型 | 模式 | 特征 | 典型公司 |
  |------|------|------|----------|
  | **🐄 奶牛型**（最佳） | `+ -` 或 `+ - -` | 主业强劲造血，投资扩张+分红回购 | 贵州茅台 |
  | **🐂 蛮牛型** | `+ - +` | 主业造血，但投资远超现金流需融资补血 | 高速扩张期公司 |
  | **🧚 妖精型** | `- +` | 主业不赚钱，靠变卖资产或投资收益维持 | 投资控股公司 |
  | **🐄 病牛型** | `- + +` | 主业失血，靠卖资产+借款度日 | 陷入困境的公司 |
  | **🃏 骗吃型** | `- - +` | 主业失血+疯狂投资，完全靠外部输血 | 商业模式未跑通公司 |

## 📊 财务字段映射表

### 🏦 三市场财务字段映射说明

财报检查清单主要基于 A 股市场编写，但可以通过字段映射应用于 A 股、港股、美股三个市场。


#### 📋 字段映射原则

1. **功能性映射**: 按财务功能（资产、负债、利润等）进行分类
2. **市场适配**: 每个市场提供对应的字段名称
3. **查询方式**: 使用 FastAPI 的 `/financial/query` 查询数据
4. **可用字段验证**: 通过以下 API 端点获取可用字段列表
   - **财务指标**: `GET /api/v1/financial/fields/a_stock/a_stock_indicators` (25个字段)
   - **资产负债表**: `GET /api/v1/financial/fields/a_stock/a_financial_statements` (75个字段)
   - **利润表**: `GET /api/v1/financial/fields/a_stock/a_financial_statements` (46个字段)
   - **现金流量表**: `GET /api/v1/financial/fields/a_stock/a_financial_statements` (72个字段)

#### 📊 字段可用性统计

- **可量化项目：19个** (约占61%)
- **无法量化项目：12个** (约占39%)

##### 主要字段缺失
- **商誉**：在字段清单中未找到，可能包含在"其他非流动资产"中
- **开发支出**：未在字段清单中找到
- **累计折旧**：无法直接获得固定资产净值
- **其他业务收入**：与主营业务收入合并为"营业收入"

##### 替代方案
- **坏账准备** → 使用"资产减值损失"+"信用减值损失"
- **存货跌价准备** → 使用"资产减值损失"
- **净资产收益率** → 财务指标表直接提供
- **税前利润** → 使用"利润总额"

### 🔍 使用示例

#### 示例1：货币资金安全性分析（三地市场）

```python
import httpx

# A股分析
response = httpx.get("http://localhost:8000/api/v1/financial/query", params={
    "market": "a_stock",
    "query_type": "a_financial_statements",
    "symbol": "SH600519",
    "start_date": "2023-01-01",
    "end_date": "2023-12-31",
    "fields": "货币资金,交易性金融资产,短期借款,长期借款"
})
data = response.json()["data"]["balance_sheet"]

# 计算货币资金安全比率
安全比率 = (data['货币资金'] + data['交易性金融资产']) / (data['短期借款'] + data['长期借款']) * 100
print(f"A股安全比率: {安全比率:.2f}%")

# 港股分析
response = httpx.get("http://localhost:8000/api/v1/financial/query", params={
    "market": "hk_stock",
    "query_type": "hk_financial_statements",
    "symbol": "00700",
    "start_date": "2023-01-01",
    "end_date": "2023-12-31",
    "fields": "现金及等价物,交易性金融资产(流动),短期贷款,长期贷款"
})
data = response.json()["data"]["balance_sheet"]

# 计算港股安全比率
安全比率 = (data['现金及等价物'] + data['交易性金融资产(流动)']) / (data['短期贷款'] + data['长期贷款']) * 100
print(f"港股安全比率: {安全比率:.2f}%")
```

#### 示例2：现金流类型分析（三地市场）

```python
# A股现金流类型
response = httpx.get("http://localhost:8000/api/v1/financial/query", params={
    "market": "a_stock",
    "query_type": "a_financial_statements",
    "symbol": "SH600519",
    "start_date": "2023-01-01",
    "end_date": "2023-12-31",
    "fields": "经营活动产生的现金流量净额,投资活动产生的现金流量净额,筹资活动产生的现金流量净额"
})
data = response.json()["data"]["cash_flow"]

经营现金流符号 = "+" if data['经营活动产生的现金流量净额'] > 0 else "-"
投资现金流符号 = "+" if data['投资活动产生的现金流量净额'] > 0 else "-"
筹资现金流符号 = "+" if data['筹资活动产生的现金流量净额'] > 0 else "-"

现金流类型 = f"{经营现金流符号}{投资现金流符号}{筹资现金流符号}"
print(f"A股现金流类型: {现金流类型}")

# 判断现金流类型
if 现金流类型 in ["+--", "+-"]:
    print("🐄 奶牛型（最佳）：主业强劲造血，投资扩张+分红回购")
elif 现金流类型 == "+-+":
    print("🐂 蛮牛型：主业造血，但投资远超现金流需融资补血")
elif 现金流类型 == "-+":
    print("🧚 妖精型：主业不赚钱，靠变卖资产或投资收益维持")
elif 现金流类型 == "-++":
    print("🐄 病牛型：主业失血，靠卖资产+借款度日")
elif 现金流类型 == "--+":
    print("🃏 骗吃型：主业失血+疯狂投资，完全靠外部输血")
```
