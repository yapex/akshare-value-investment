# 📊 财报检查清单

## 一、财报获取与公司基本情况

### 1.1 公司是否按时发布财报？
- **追问：** 若无法按时发布，原因是什么？是否为危险信号？

### 1.2 公司基本情况是否了解？
- **内容：** 规模、行业地位、主营业务、发展历程等
- **追问：** 主营业务是否清晰？历史融资及分红情况如何？

### 1.3 股权结构是否清晰？
- **追问：** 主要股东及持股比例如何？有无控股股东持续大量减持的情况？

## 二、"重要提示"部分

### 2.1 会计师事务所是否发生变更？
- **追问：** 变更原因是什么？是否属于国资委要求的定期更换？若为再次更换，是否需警惕？

### 2.2 会计师出具的意见是否为"标准无保留意见"？
- **追问：** 若不是，具体是什么意见？是否应直接排除该公司？

### 2.3 高管团队是否稳定？
- **追问：** 财务总监是否频繁更换？独立董事或监事有无集体辞职等异常变动？

### 2.4 有无其他异常说明？
- **追问：** 独立董事或监事是否在"重要提示"中做出特别提醒或说明？

## 三、"资产负债表"及相关附注

### 3.1 资产类项目

#### 3.1.1 货币资金是否安全？
- **追问：** 现金及现金等价物能否覆盖有息负债？若不能，加上可迅速变现的金融资产后能否覆盖？
- **计算公式：**
  ```
  货币资金安全比率 = (现金及现金等价物 + 可迅速变现的金融资产) ÷ 有息负债 ≥ 1
  ```

#### 3.1.2 货币资金是否存在异常？
- **追问：** 余额是否远小于短期负债？是否存在大额受限资金？利息收入是否显著低于市场利率？
- **计算公式：**
  ```
  模拟利息收入 = 货币资金平均余额 × 市场常见利率
  ```

#### 3.1.3 应收票据是否健康？
- **追问：** 商业承兑汇票占比是否过高？是否突然大量出现商票？应收票据规模是否与正常经营规模匹配？

#### 3.1.4 应收账款是否健康？
- **追问：**
  - 增长是否快于收入？
  - 占总资产或收入的比例是否过大？
  - 账龄是否过长（如超一年）？
  - 周转率是否呈下降趋势或低于行业水平？
  - 有无集中或异常的应收款？
- **计算公式：**
  ```
  应收账款增长率 vs 营业收入增长率
  应收账款占总资产比例 = (应收账款 + 其他应收款 - 银行承兑汇票) ÷ 总资产
  应收账款周转率 = 营业收入 ÷ 平均应收账款
  应收账款回款周期 = 应收账款余额 ÷ 月均营业收入
  ```

#### 3.1.5 预付账款是否异常？
- **追问：** 是否大幅增长（尤其是工程款、技术采购款）？有无长期挂账？占收入或成本的比例是否大幅波动？

#### 3.1.6 其他应收款是否异常？
- **追问：** 剔除应收股利和利息后，数额是否较大？

#### 3.1.7 坏账准备计提是否合理？
- **追问：** 计提标准与同行相比如何？有无变更标准？有无异常的统一比例计提或应提未提情况？

#### 3.1.8 存货是否存在风险？
- **追问：**
  - 存货（特别是易贬值品）的减值准备计提是否充分？
  - 存货增长是否快于营业成本？
  - 存货周转率下降的同时毛利率是否异常提升？
- **计算公式：**
  ```
  存货增长率 vs 营业成本增长率
  存货周转率 = 营业成本 ÷ 平均存货
  ```

#### 3.1.9 公司是否属于财务造假高危领域？
- **追问：** 是否为农林牧渔公司？税收优惠是否多？产品是否不易核查或估值？

#### 3.1.10 长期股权投资是否存在操纵迹象？
- **追问：**
  - 有无通过小额交易改变股权分类以调节利润？
  - 交易对手方是否疑似同一控制？
  - 有无将一笔交易分拆多笔进行？

#### 3.1.11 投资性房地产的计量模式是什么？
- **追问：** 是以成本模式还是公允价值模式计量？

#### 3.1.12 固定资产折旧政策是否变更？
- **追问：** 变更原因及影响是否合理？有无账面价值极低但实际价值未降的资产（如酒窖、房产）？
- **计算公式：**
  ```
  固定资产周转率 = 营业收入 ÷ 固定资产净值
  ```

#### 3.1.13 在建工程是否异常？
- **追问：**
  - 有无已完工但迟迟不转固？
  - 有无不断超预算追加投资？
  - 工程进度与资金投入是否匹配？
  - 转固后是否很快需要大修？

#### 3.1.14 开发支出资本化是否合理？
- **追问：** 需结合行业与公司特点判断资本化政策的合理性。

#### 3.1.15 商誉是否过大？
- **追问：**
  - 收购作价是否公允？
  - 大额商誉是否使负债率堪忧？
  - 减值测试披露是否充分（有无过程披露）？
  - 收购并出售大额商誉资产的交易是否可疑？
- **计算公式：**
  ```
  调整后负债率 = (总负债 + 商誉) ÷ 总资产
  ```

#### 3.1.16 长期待摊费用数额是否过大？
- **追问：** 是否可能存在不恰当的费用资本化？

### 3.2 负债与权益类项目

#### 3.2.1 有息负债占比及成本如何？
- **追问：**
  - 有息负债占总资产比例与行业龙头相比如何？
  - 借款利率是否显著高于市场水平？
  - 是否在货币资金充裕时仍高息借款？
- **计算公式：**
  ```
  有息负债占比 = 有息负债 ÷ 总资产
  借款利率 = 利息支出 ÷ 平均有息负债
  ```

#### 3.2.2 应付账款/票据是否异常大增？
- **追问：** 是行业地位提升所致，还是资金链紧张的表现？

#### 3.2.3 预收款项数额是否略高？
- **追问：** 是否存在推迟确认收入、隐藏利润的可能？

#### 3.2.4 应付职工薪酬是否透露关键信息？
- **追问：**
  - 高管及核心人员薪酬与竞争力是否匹配？
  - 员工平均薪酬是否合理（历史与同业对比）？
  - 公司盈利对工资上调的敏感性如何（工资黏性）？
  - 有无通过预提薪酬平滑利润的迹象（结合递延所得税资产看）？
- **计算公式：**
  ```
  员工平均薪酬 = 应付职工薪酬（本期增加额） ÷ 员工总数
  工资黏性测试：上调后净利润 = 原净利润 - (员工总数 × 人均薪酬 × 上调比例)
  ```

#### 3.2.5 有无隐藏的良性债务？
- **追问：** 专项应付款、递延收益余额是否较大？

#### 3.2.6 库存股回购的目的是什么？
- **追问：** 是用于注销（价格是否合理）还是股权激励（费用分摊如何）？

### 3.3 整体分析指标

#### 3.3.1 公司成长性与营运能力如何？
- **追问：** 总资产增长率、净资产增长率、总资产周转率与行业龙头及自身历史相比如何？
- **计算公式：**
  ```
  总资产增长率 = (期末总资产 - 期初总资产) ÷ 期初总资产
  净资产增长率 = (期末净资产 - 期初净资产) ÷ 期初净资产
  总资产周转率 = 营业收入 ÷ 平均总资产
  ```

## 四、"利润表"及相关附注

### 4.1 主营业务是否聚焦？
- **追问：** 主营业务是否过于多元化？收入确认原则（特别是完工百分比法）是否清晰？

### 4.2 营业收入增长是否健康？
- **追问：**
  - 增速是否高于行业平均？
  - 增长来源于需求、份额还是提价？
  - 对大客户依赖度是否过高？
- **计算公式：**
  ```
  营业收入增长率 = (本期营业收入 - 上期营业收入) ÷ 上期营业收入
  ```

### 4.3 有无可疑的收入或关联交易？
- **追问：**
  - 是否销售陌生产品？
  - 是否突现大额陌生客户交易？
  - 关联交易的必要性、定价是否合理？

### 4.4 毛利率是否异常？
- **追问：** 与同行相比是否显著偏离？历史变动原因是成本还是售价？
- **计算公式：**
  ```
  毛利率 = (营业收入 - 营业成本) ÷ 营业收入
  ```

### 4.5 费用控制是否有效？
- **追问：**
  - 管理费用增幅是否超过营收？
  - 销售与管理费用率与同行相比如何？
  - 三费（费用率）占毛利润的比例是多少？
  - 研发费用占比是否过高（若不懂则谨慎）？
- **计算公式：**
  ```
  费用率 = (销售费用 + 管理费用 + 财务费用*) ÷ 营业总收入
  费用占毛利润比 = (销售费用 + 管理费用 + 财务费用) ÷ 毛利润
  ```
  *财务费用为正则加，为负则减

### 4.6 资产减值损失有无操纵痕迹？
- **追问：** 有无不提、超额计提、洗大澡等迹象？

### 4.7 营业利润增长质量如何？
- **追问：** 增速是否不低于营收增速？营业利润率变动原因是什么？
- **计算公式：**
  ```
  营业利润增长率 vs 营业收入增长率
  营业利润率 = 营业利润 ÷ 营业收入
  ```

### 4.8 其他收益是否可靠？
- **追问：**
  - 其他业务收入占比是否突然大增？
  - 营业外收入净额是否过大？
  - 利润是否主要依赖营业外收入？
- **计算公式：**
  ```
  其他业务收入占比 = 其他业务收入 ÷ 营业收入
  营业外收入净额 = 营业外收入 - 营业外支出
  ```

### 4.9 净利润含金量高吗？
- **追问：** 经营现金流净额连续数年是否大于净利润？
- **计算公式：**
  ```
  净利润含金量 = 经营活动产生的现金流量净额 ÷ 净利润
  ```

### 4.10 核心收益率指标如何？
- **追问：** 净资产收益率(ROE)是否连续大于15%？总资产收益率(ROA)历史与同业对比如何？
- **计算公式：**
  ```
  净资产收益率(ROE) = 净利润 ÷ 平均净资产
  总资产收益率(ROA) = 净利润 ÷ 平均总资产
  ```

### 4.11 公司属于重资产还是轻资产？
- **追问：** 生产资产占比如何？其税前回报率是否显著高于社会平均资本回报率？
- **计算公式：**
  ```
  生产资产占比 = (固定资产 + 在建工程 + 工程物资 + 无形资产中的土地) ÷ 总资产
  生产资产回报率 = 税前利润总额 ÷ 生产资产
  ```

## 五、"现金流量表"及相关附注

### 5.1 经营活动现金流是否优秀？
- **追问：**
  - 是否连续数年大于净利润及折旧摊销？
  - 是否持续优于同行？
  - 若为正，是否主要靠应付项目增加？
  - 若持续为负却有巨额利润，是否报警？
- **计算公式：**
  ```
  经营现金流健康度 = 经营活动产生的现金流量净额 > (净利润 + 折旧与摊销)
  ```

### 5.2 投资活动现金流是否健康？
- **追问：**
  - 剔除理财后，净流出是否主要用于扩张？
  - 投资回报率是否高于资金平均成本？
  - 现金流入是否大量来自变卖资产？
  - 流出是否严重超预算？
- **计算公式：**
  ```
  投资回报率 ≈ (投资收益 + 利息收入) ÷ 平均投资资产
  ```

### 5.3 筹资活动现金流是否异常？
- **追问：**
  - 在需要现金时是否大量还贷（银行缩贷）？
  - 借款利率是否显著偏离市场？
  - 是否同时存在高息借款和低收益理财？

### 5.4 公司整体现金流类型是什么？
- **追问：**
  - 是否为"正负负"（经营流入，投资、筹资流出）或"正正负"（经营、投资流入，筹资流出）型？
  - 投资流出是用于扩张还是维持？
  - 筹资流出是用于分红还是还债？
  - 投资流入是否可持续、是否变卖核心资产？

## 六、董事会报告与附注

### 6.1 董事会报告是否提供了关键信息？
- **追问：**
  - 在建项目变更原因及金额变动是否合理？
  - 管理层对未来的分析与讨论，与同行及公司历史相比，是否显示出专业水平和诚信？

### 6.2 是否已系统浏览所有财务报表附注？
- **追问：** 附注中是否隐藏了未在主要报表项目中充分体现的重要信息或风险？

## 📊 财务字段映射表

### 🏦 三市场财务字段映射说明

财报检查清单主要基于 A 市场编写，但可以通过字段映射应用于 A 股、港股、美股三个市场。

#### 📋 字段映射原则

1. **功能性映射**: 按财务功能（资产、负债、利润等）进行分类
2. **市场适配**: 每个市场提供对应的字段名称
3. **查询方式**: 使用 MCP 工具的 `get_available_fields()` 方法获取完整字段列表

### 📝 使用方法

#### 1. 获取字段列表
```python
from akshare_value_investment.mcp.tools.financial_query_tool import FinancialQueryTool

tool = FinancialQueryTool()

# 获取A股财务指标字段
a_stock_fields = tool.get_available_fields('a_stock', 'a_stock_indicators')
print(f"A股字段: {a_stock_fields['available_fields'][:10]}")

# 获取港股财务指标字段
hk_stock_fields = tool.get_available_fields('hk_stock', 'hk_stock_indicators')
print(f"港股字段: {hk_stock_fields['available_fields'][:10]}")

# 获取美股财务指标字段
us_stock_fields = tool.get_available_fields('us_stock', 'us_stock_indicators')
print(f"美股字段: {us_stock_fields['available_fields'][:10]}")
```

#### 2. 使用字段进行查询
```python
# 查询A股数据
response = tool.query_financial_data(
    market='a_stock',
    query_type='a_stock_indicators',
    symbol='SH600519',
    fields=['报告期', '净利润', '毛利率'],
    start_date='2023-01-01',
    end_date='2023-12-31'
)

# 查询美股数据
response = tool.query_financial_data(
    market='us_stock',
    query_type='us_stock_indicators',
    symbol='AAPL',
    fields=['REPORT_DATE', 'PARENT_HOLDER_NETPROFIT', 'GROSS_PROFIT_RATIO'],
    start_date='2023-01-01',
    end_date='2023-12-31'
)
```

#### 3. 计算财务指标
```python
# A股 ROE计算示例
roa = data['净利润'] / data['资产总计']

# 美股指标直接获取
roe = data['ROE']  # 美股提供现成的ROE字段
```
